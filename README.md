# js_challenge_jagaad_Dmytro-Puzyrenko
### The code built by the guide: https://wiki.jagaad.it/en/recruiting/frontend-coding-challenge#Guidelines

## Run Specs
- macOS Big Sur  
- nodejs v15.5.1
- npm v7.3.0
- vue/cli v4.5.10

## Installation
```sh
npm install
```
OR 
```sh
yarn
```
## For dev environment:
```sh
npm run dev
```
OR
```sh
yarn run dev
```

## For prod environment:
```sh
npm run build
```
OR
```sh
yarn run build
```

## Testing tools:
### e2e testing
```sh
npm run test:e2e:run
```
OR
```sh
yarn run test:e2e:run
```

### unit testing
```sh
npm run lint
```
OR
```sh
yarn run lint
```
